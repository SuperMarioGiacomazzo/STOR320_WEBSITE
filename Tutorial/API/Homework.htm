Network Communication Homework

 

Using the file place_locs.csv found on the course website, construct the file place_locs_with_latlon_{studentname}.csv. For example, my file would be named place_locs_with_latlon_marshallmarkham.csv.

The file should contain all of the original columns with two additional columns, lat and lon appended to the right of the original data. You can geocode the addresses found in place_locs.csv using the API documented at: https://www.census.gov/data/developers/data-sets/Geocoding-services.html

There are also instructions on API usage at: https://geocoding.geo.census.gov/geocoder/Geocoding_Services_API.pdf

Hints:

1. There is a batch geocoding api available, but if you feel up to the challenge try using the single address api on a loop (obviously we would not do this IRL, but it is good practice working iteratively)

2. Use benchmark=9 as the benchmark parameter
